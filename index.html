Disciplina de Desenvolvimento Web

Este trabalho foi densenvolvido com o intuito de demontrar os conhecimentos em Desenvolvimento Web aprendidos fora de sala de aula.

A ideia até o momento é desenvolver uma página de formulário de pesquisa. Com isto, espera-se ser possível aprender e demonstrar conhecimentos em HTML, CSS, JavaScript e etc.

Passos para o desenvolivmento:

  1. Estudar HTML
  2. Desenvolver o esqueleto de uma página em HTML
  3. Estudar CSS, Bootstrap
  4. Estudar JavaScript
  5. Implementar funcionalidades em JavaScript
  6. Ligar a um banco de dados
